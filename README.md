# HireVision AI – Intelligent Recruitment Analytics Platform

HireVision AI is an AI-powered recruitment platform that automates resume screening, candidate ranking, and hiring analytics.  
The system leverages Natural Language Processing (NLP) and machine learning techniques to help recruiters efficiently identify the best candidates.

## 🚀 Features

- Automated Resume Screening
- Candidate Ranking based on Job Description similarity
- Skill Extraction from resumes
- ATS Compatibility Scoring
- AI Resume Feedback generation
- Interview Question Generator
- Hiring Analytics Dashboard
- Skill Distribution Visualization
- AI Resume Chatbot (RAG-based) for recruiter queries


## 🧠 How It Works

1. Recruiter uploads candidate resumes.
2. System extracts resume text and identifies candidate skills.
3. NLP similarity techniques compare resumes with the job description.
4. ATS scores and candidate rankings are generated.
5. Analytics dashboards provide insights about candidates.
6. Recruiters can interact with an AI chatbot to query resume details.

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Frontend:** Streamlit  
- **Machine Learning / NLP:** Scikit-learn, TF-IDF  
- **Data Processing:** Pandas
